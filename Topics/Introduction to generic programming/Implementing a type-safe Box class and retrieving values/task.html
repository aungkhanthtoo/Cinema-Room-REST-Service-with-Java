Create a type-safe Box class that has a single instance variable for the type it's defined with, alongside an appropriate constructor. Implement a method named get that will return the value. Note that the box should be able to hold any data type. The input will be a single line of either string, integer or float and your output should be the same value.

Sample Output:
5
